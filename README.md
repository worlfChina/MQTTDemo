# MqttDemo
适应于IOT场景的轻型网络框架。  
观察者模式，订阅/发布消息机制。  

## 优点：  
1.支持TCP连接，及SSL加密；  
2.支持断线缓存及断线重连；  
3.可适应于极差的网络环境，让IOT-Android开发工程师更专注于业务开发。
